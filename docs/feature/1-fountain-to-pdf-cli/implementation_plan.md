# 実装計画: Fountain-to-PDF CLI ツールの導入

## 概要
Fountain 形式の脚本ファイルを PDF に変換するコマンドラインインターフェース (CLI) ツールを実装します。単一ファイルで完結し、コマンドライン引数で入出力を指定可能にします。

## 構成
- **言語**: Python 3.9+
- **使用ライブラリ**: 
  - `playscript`: Fountain -> PDF 変換のコア（既存）
  - `argparse`: コマンドライン引数の解析
- **ファイル**: `src/pdf_converter.py`

## 実装内容
1.  **引数解析の追加**: `argparse` を導入し、入力ファイルパスと出力ファイルパスを指定可能にします。
2.  **デフォルト動作**: 入力ファイルが指定されない場合はヘルプを表示、出力ファイルが指定されない場合は入力ファイル名から推測（`.fountain` -> `.pdf`）または標準的な場所に出力します。
3.  **エラーハンドリング**: ファイルが存在しない場合や、変換エラーが発生した場合に適切なメッセージを表示します。
